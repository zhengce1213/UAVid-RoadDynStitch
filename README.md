# UAVid-RoadDynStitch

UAVid-RoadDynStitch is a dynamic UAV road image-pair subset for studying vehicle- and shadow-aware road image stitching, seam selection, and image source selection.

This subset is derived from the UAVid dataset. We select UAV road image pairs containing roads, vehicles, and dynamic occlusions from UAVid, and generate vehicle/shadow masks using a fine-tuned vehicle instance and shadow segmentation model. These masks are used for semantic-guided fusion in image stitching experiments.

This repository is intended for non-commercial academic research only.

## Dataset Overview

In low-altitude UAV road image stitching, moving vehicles, vehicle shadows, road markings, road texture variations, building occlusions, and exposure differences can significantly affect stitching quality. These factors may lead to seams crossing vehicles, vehicle residuals, ghosting artifacts, shadow residuals, and discontinuous road textures.

UAVid-RoadDynStitch focuses on vehicle and shadow occlusions in UAV road scenes. It is suitable for research on:

- UAV road image stitching;
- vehicle- and shadow-aware seam selection;
- source selection in dynamic scenes;
- semantic-mask-guided image fusion;
- stitching quality evaluation for road inspection scenarios.

## Dataset Contents

This repository contains:

```text
UAVid-RoadDynStitch/
  images/
    input1/
    input2/
  masks/
    input1/
    input2/
  splits/
    test_pairs.csv
  assets/
    duibi2.drawio.png
  README.md
  LICENSE
```

where:

- `images/input1`: the first image in each image pair;
- `images/input2`: the second image in each image pair;
- `masks/input1`: the vehicle/shadow mask corresponding to the first image;
- `masks/input2`: the vehicle/shadow mask corresponding to the second image;
- `splits/test_pairs.csv`: the mapping between released image pairs and their original sequence/file names;
- `assets/duibi2.drawio.png`: a qualitative comparison figure used in the paper.

The vehicle/shadow masks are generated by a fine-tuned vehicle instance and shadow segmentation model, and are used to support semantic-aware source selection and seam guidance in image stitching.

## Relationship to UAVid

This subset is a derived dataset built upon the UAVid dataset. The original images come from UAVid. We only select and reorganize road scenes suitable for dynamic image stitching research, and additionally provide vehicle/shadow masks.

UAVid official website:

https://uavid.nl/

UAVid paper DOI:
